    malloc()

    • 英语：memory allocation

    • 头文件： <stdlib.h>

    • 函数原型

      void* malloc (size_t size);//分配成功返回指针，分配失败返回NULL

      返回值为void*要注意使用时强制转换类型

      例如

      char*ptr = (char*)malloc(10);//分配10个字节的内存空间，用来存放字符

    • 释放内存

      free(*变量名*)；

    • 注意事项

      分配内存空间时储存字符串时，多分配一位储存字符串结尾的\0标识符

    • 参考

    calloc()

    • 英语：clear allocation
    • 头文件： <stdlib.h>
    • 函数原型

      void* calloc (size_t num, size_t size);//分配num个长size的空间且初始化为0，成功返回指针，失败返回NULL

    size_t

    • 表示C语言中任何对象所能达到的最大长度，在32位系统中位unsigned int，在64位系统中为unsigned long

    new()

    • 头文件： <new>

    • 用法

      int *p = new int;
      int p =*new int;
      定义单个变量
      int *p = new int(x);
      定义变量并初始化为x;
      delete p;
      释放单个变量空间
      int*a = new int[5];
      定义数组，a可直接当作数组使用
      delete []a;
      释放数组空间
      定义类型可以是用户自定义的类型
